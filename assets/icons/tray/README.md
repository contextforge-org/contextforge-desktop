# Tray Icons

This directory contains system tray icons for different platforms.

## Required Icons

### macOS
- `iconTemplate.png` (16x16) - Standard resolution
- `iconTemplate@2x.png` (32x32) - Retina resolution
- Use "Template" suffix for automatic dark mode support

### Windows
- `icon.ico` - Multi-resolution ICO file (16x16, 32x32, 48x48, 256x256)

### Linux
- `icon.png` (16x16) - Standard resolution
- `icon@2x.png` (32x32) - High DPI

## Creating Icons

You can create placeholder icons or use your own designs. For production:
1. Design a simple, recognizable icon
2. Export in required sizes
3. For macOS, use monochrome template images
4. For Windows, create a multi-size ICO file
5. Test on all platforms

## Placeholder Icons

Temporary placeholder icons are included. Replace with your actual app icons before production.
